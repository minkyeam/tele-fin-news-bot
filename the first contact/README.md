# Little Hacker's Quest (LHQ) - Voice Edition

6세 아이를 위한 웹 기반 CLI 놀이 게임입니다. 텍스트를 아직 읽지 못해도, 한국어 TTS 안내를 들으며 알파벳을 찾아 입력할 수 있습니다.

## 실행 방법

1. `index.html`을 브라우저에서 엽니다.
2. 큰 `시작!` 버튼을 누르면 음성 안내가 시작됩니다.

## 구현된 기능

- 레벨 진입 시 상황 설명 + 철자 가이드 TTS 자동 재생
- 정답 입력 시 칭찬/상황 변화 TTS 재생
- 오답 입력 시 재도전 안내 TTS
- `키 입력 알파벳 읽기` 토글 제공
- 3개 레벨 시나리오 반영
  - Lv.1 `on`
  - Lv.2 `feed`
  - Lv.3 `wake`

## 브라우저 참고

- Web Speech API가 지원되는 최신 Chrome/Edge에서 가장 안정적으로 동작합니다.
- 브라우저 정책 때문에 첫 음성 재생은 사용자 클릭(시작 버튼) 이후 동작하도록 구성되어 있습니다.
